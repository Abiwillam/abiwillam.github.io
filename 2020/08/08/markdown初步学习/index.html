1. 标题

支持一到六级标题，一个井号（#）算一级
eg.

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题

2. 字体

·加粗

文本用两个星号左右包围
eg.
加粗

·斜体

文本左右用一个星号包围起来
eg.
斜体

·斜体加粗

文本左右用三个星号包围起来
eg.
*斜体加粗**

·删除线

文字左右用两个波浪线包围起来
eg.
删除线

3. 引用

在文字前加大于号（>）即可，可嵌套，嵌套上一条的条件是要在上一条的基础上多加一个大于号
eg.

这是一条引用内容

这是上一条的嵌套

老俄罗斯套娃🌶

4. 分隔线

三个或三个以上的- ，*都可以
eg.

  • 三个（及以上）-
  • 三个（及以上）*

效果相同

5. 图片

语法：

![图片alt](图片地址 ‘’图片title’’)

图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加
没有例子（暂时）

6. 超链接

语法

[超链接名](超链接地址 "超链接title")
title可加可不加
新标签页打开需用html的a标签
<a href="超链接地址" target="_blank">超链接名</a>

eg.

[百度](https://www.baidu.com)
<a href="(https://www.baidu.com" target="_blank">百度</a>

百度
百度

7. 代码(块)

·单行代码

代码之间分别用一个反引号(英文状态下Tab上面的键包起来
eg.
print("Hello,world!")

·代码块

代码之间用三个反引号包起来
eg.

funtion main(){
    println("Hello,world!")
}

8. 列表

·无序列表

用-，+，*都可以作为无序列表，符号与文字之间需要有空格
eg.

- 列表
+ 列表
* 列表
  • 列表
  • 列表
  • 列表

·有序列表

数字加点即可，点后需有一个空格
eg.

1. 列表1
2. 列表2
3. 列表3
  1. 列表1
  2. 列表2
  3. 列表3

·列表嵌套

上一级和下一级之间需敲四个空格
eg.

+ 一级无序列表
    1. 二级有序列表1
    2. 二级有序列表2
    3. 二级有序列表3

1.一级有序列表
    + 二级无序列表1
    + 二级无序列表2
    + 二级无序列表3
  • 一级无序列表
    1. 二级有序列表1
    2. 二级有序列表2
    3. 二级有序列表3
  1. 一级有序列表
    • 二级无序列表1
    • 二级无序列表2
    • 二级无序列表3

9. 表格

语法

表头|表头|表头
---|---|---
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略

eg.
样式|名称
-|-
1|阿拉伯数字
一|中国汉字
One|英语

p.s. 不支持？vscode显示正常emmmm
